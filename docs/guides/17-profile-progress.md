# 17 — Profile & Progress

The profile system tracks your journey across sessions. It's entirely local — nothing is shared or uploaded.

---

## XP and levels

You earn **XP (experience points)** by completing sessions. XP accumulates across all sessions and unlocks levels.

**Base XP per session:**
- Completed session: 20 XP
- Interrupted session: 8 XP
- +1 XP per minute (up to 20 bonus XP)
- +12 XP for zero attention losses
- +4 XP if haptic track active
- +3 XP if visualization block active
- +2 XP for scenes, rules, intensity ramp

**20 levels total:** Initiate → Curious → Attuned → Focused → Steady → Devoted → Controlled → Refined → Trained → Practiced → Disciplined → Immersed → Surrendered → Shaped → Conditioned → Dedicated → Elite → Master → Transcendent → Apex

---

## Daily quests

Three quests are generated each day. They reset at midnight. Complete them for bonus XP.

**Example quests:**
- Complete any session (+20 XP)
- Complete a 10-minute session (+25 XP)
- Complete a session with no attention losses (+35 XP)
- Complete a session with haptic feedback active (+30 XP)
- Complete 2 sessions today (+40 XP)

Quest XP is awarded in the post-session summary.

---

## Achievements

71 achievements across 8 categories. 19 are hidden — you discover them by doing unusual things.

| Category | Examples |
|----------|---------|
| First Steps | First session, first quest, first level up |
| Consistency | 2-day streak, monthly visitor, 3 months active |
| Sessions | 3, 5, 10, 20, 30, 50 completed sessions |
| Endurance | 30 min total, 1 hour total, 10-minute single session |
| Focus | 80% focus time, perfect session (zero losses) |
| Craft & Exploration | Use TTS, viz, haptic, rules, variables, all modes |
| Quests | 5, 10, 25, 50 quests completed |
| Levels | Reach levels 3, 5, 7, 10, 15, 20 |

---

## Viewing your profile

Click the avatar icon in the top bar to open the profile panel.

The panel shows:
- Avatar (click to cycle through emojis)
- Display name
- XP bar with level and progress
- Daily quests with completion status
- Achievement grid organized by category

---

## Notifications

Four types of progress notifications appear after sessions:

| Notification | Trigger |
|-------------|---------|
| XP earned | Any completed session |
| Level up | Reaching a new level |
| Achievement unlocked | Earning a new achievement |
| Quest completed | Completing a daily quest |

Each can be individually toggled in **Settings → Display / HUD → Progress notifications**.

---

## Resetting

**Clear profile:** Settings → System → Clear Profile & History

This permanently erases XP, achievements, session count, streaks, and history. Two confirmation dialogs required.
