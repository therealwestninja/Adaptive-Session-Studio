# 10 — Visualizations

Visualization blocks overlay animated graphics on the session during playback. They're designed for hypnotic induction and depth work.

---

## The five visualization types

| Pattern | Description | Best for |
|---------|-------------|----------|
| 🌀 **Spiral** | Rotating Archimedes spiral | Classic induction, eye fixation |
| 〰 **Pendulum** | Harmonograph-style trace | Entrancing, organic movement |
| ⭕ **Tunnel** | Concentric rings rushing inward | Depth induction, falling sensation |
| 💗 **Pulse** | Expanding rings | Heartbeat sync, rhythm work |
| 🌪 **Vortex** | Multi-arm rotating star | Peak scenes, surrender work |

---

## Adding a visualization block

1. Create a new **Viz** block in the timeline
2. Set start time and duration
3. In the inspector:
   - Choose the pattern
   - Set speed (0.25 = very slow, 4.0 = fast)
   - Choose the primary color
   - Set opacity (how much it overlays the background)

Viz blocks can run simultaneously with text blocks — the visualization plays behind the text.

---

## Effective use

**Induction sessions:** Start with a slow spiral or tunnel during the settling scene to give the eyes something to fix on.

**Depth scenes:** Switch to a vortex or faster tunnel speed during the peak scene to intensify the experience.

**Recovery scenes:** Remove visualizations or reduce opacity — returning from visualization to a blank screen signals the end of depth work.

**Breath sync with pulse:** Set the pulse rate to match a breathing cue ("in for 4, out for 4") for somatic grounding.
