# 12 — Session Modes

Session modes are one-click presets that configure the session's rules, intensity ramp, and pacing for a specific experience type.

---

## The eight modes

### Exposure Therapy
Gradual desensitization approach. Pauses on attention loss with a fallback intensity reduction on overload. Good for new users or early sessions.

### Mindfulness / Focus
Meditation-oriented. Pauses on distraction. Reduces intensity on drift. Designed for sustained attention practice.

### Deep Focus
More demanding version of mindfulness. Three rules: pause on break, escalate on peak engagement, recover from disengagement. For experienced practitioners.

### Free Run
No rules, no automatic adaptation. Pure manual control. The session plays through exactly as designed, with the operator adjusting live controls.

### Guided Induction
Follows a standard induction arc. Slower pacing, engagement-based ramp. Works well with visualization blocks.

### Behavioral Conditioning
Adaptive reward and correction. Three rules: reward sustained focus (increase intensity), redirect on drift (next scene), escalate on peak (push intensity up). Designed for training sessions.

### Operator-Led Training
Manual control focus. Configures the session for a two-person setup where an operator controls everything and the subject responds.

### Deep Surrender
Maximum surrender experience. Exponential ramp to high intensity. No automatic rules — the session escalates continuously. For experienced, fully consenting participants.

---

## Applying a mode

Use the **Mode selector** dropdown in the toolbar, or open the **Modes** tab in the inspector.

When you apply a mode, it:
1. Replaces the current rules with the mode's preset rules
2. Sets the intensity ramp configuration
3. Sets pacing settings
4. Sets any mode-specific parameters

**This overwrites your existing rules.** If you've customized rules, apply the mode first, then modify from there.

---

## Customizing after applying

Modes are starting points. After applying one, you can:
- Add, remove, or modify rules
- Adjust ramp curves
- Change pacing settings
- Save the customized session for reuse
