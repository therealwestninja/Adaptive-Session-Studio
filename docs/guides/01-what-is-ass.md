# 01 — What is Adaptive Session Studio?

## The short version

Adaptive Session Studio is a desktop application that runs entirely in your web browser. It lets you design and play structured adult sessions that combine:

- **Written prompts** shown on screen at exact times
- **Text-to-speech narration** read aloud from scripts you write
- **Background audio and video** playing under your content
- **Haptic device control** — FunScript patterns sent to devices via Intiface Central
- **Behavioral automation** — rules that respond to your attention and engagement in real time
- **Hypnotic visuals** — animated patterns for induction and depth work

Everything is local. No internet required after the first setup. No accounts. No data leaves your machine.

---

## Who is this for?

**Solo practitioners** who want structured, automated sessions they can run themselves without a partner.

**Operators and partners** who want to run sessions for someone else with live controls — adjusting intensity, speed, and injecting haptic patterns in real time.

**Creators** who want to build reusable session templates and share them as `.assp` files.

---

## What can it actually do?

### Content and narration
Write timed text overlays that appear during playback. Record or type TTS scripts that are read aloud using your browser's built-in speech synthesis. Import background audio (music, ambience) and video that loops throughout the session.

### Haptic device control
Connect any Buttplug.io/Intiface-compatible device. Load a FunScript file with precise time-coded positions, or use one of the 11 built-in motion patterns. Edit patterns on an interactive canvas. The device follows the session timeline automatically.

### Adaptive responses
Use the rules engine to make sessions respond dynamically. If the attention tracker detects you've looked away, the session can pause, reduce intensity, or switch to a recovery scene — automatically. Rules fire based on attention, engagement, intensity, speed, device load, session time, and loop count.

### Scenes and structure
Divide sessions into named phases — Settle, Drop, Depth, Rise — each with its own intensity profile and optional automatic transitions.

### AI session generation
Give Claude a text description of what you want ("a 5-minute grounding session with three building scenes and a gentle TTS voice") and it generates the session structure for you.

---

## What it is NOT

- Not a video player (it plays short background clips, not full films)
- Not a device firmware tool (it talks to Intiface Central, which handles the device)
- Not a cloud service (nothing is stored anywhere except your own browser's local storage)
- Not commercial software (personal use only)

---

## The name

**Adaptive Session Studio** → **ASS**.

Sessions save as `.assp` files and subtitles use `.ass` format.

This was not planned. It was noticed, found funny, and kept.
