# 20 — Settings Reference

Complete reference for every setting. Open with **Ctrl+,** or the gear icon.

---

## Appearance

| Setting | Description |
|---------|-------------|
| Theme | Built-in color theme: Midnight, Ember, Moss, Violet, Dusk, Slate |
| Background color | Custom background color (hex) |
| Accent color | Custom accent/highlight color (hex) |
| Text color | Custom text color (hex) |
| Overlay font | Font family for text overlays |
| Font size | Default font size for text blocks |
| Stage blur | Background blur amount (px) |

---

## Playback

| Setting | Description |
|---------|-------------|
| Master volume | Global audio volume (0–200%) |
| Loop mode | Off / Loop forever / Loop N times |
| Loop count | Number of loops when loop mode is active |
| Playlist audio volume | Volume multiplier for background audio tracks |
| Playlist video volume | Volume multiplier for background video tracks |

---

## Display / HUD

| Setting | Description |
|---------|-------------|
| Show metric bars | Show intensity/speed/attention bars in HUD |
| Show scene name | Show current scene name in HUD |
| Show macro slots | Show macro slot indicators in HUD |
| Show variables | Show variable values in HUD |
| Show keyboard hints | Show keyboard shortcut reminders |
| HUD auto-hide | Seconds before HUD fades (0 = never) |
| XP earned toast | Show "+N XP" toast after sessions |
| Level-up announcement | Show level-up toast |
| Achievement unlocked | Show achievement toast |
| Quest completed | Show quest completion toast |
| Sensor Bridge URL | WebSocket address for the live panel |
| Auto-connect on start | Connect sensor bridge when play begins |

---

## FunScript

| Setting | Description |
|---------|-------------|
| Device WebSocket | Intiface Central address (default: ws://localhost:12345) |
| Playback speed | Global FunScript speed multiplier |
| Invert positions | Flip all FunScript values (0↔100) |
| Range cap | Maximum position sent to device (0–100) |

---

## Macros

Assign macros to the five keyboard slots (1–5). Select a macro from the library for each slot.

---

## Subtitles

| Setting | Description |
|---------|-------------|
| Default style | Override subtitle font, size, color, position |
| Enable by default | Auto-enable new subtitle tracks on import |

---

## Webcam

| Setting | Description |
|---------|-------------|
| Enabled | Turn attention tracking on/off |
| Warmup period | Seconds before tracking starts (default: 1.5s) |
| Loss threshold | Minimum attention value before "lost" event |
| FaceDetector | Browser API availability status |

---

## Safety

| Setting | Description |
|---------|-------------|
| Max intensity | Hard cap on intensity (0–2, default: 2.0) |
| Max speed | Hard cap on speed (0.25–4, default: 4.0) |
| Warn above | Show warning indicator when intensity exceeds this (default: 1.5) |

---

## Advanced

| Setting | Description |
|---------|-------------|
| JSON editor | View and edit the raw session JSON |
| Apply JSON | Replace current session with pasted JSON |
| Sensor Bridge | Connect/disconnect the biometric bridge |
| Ramp settings | Intensity ramp curve configuration |
| Pacing settings | Dynamic speed pacing configuration |

---

## System

| Setting | Description |
|---------|-------------|
| Restart Tutorial | Re-run the first-launch onboarding |
| Clear Profile & History | Permanently erase all profile data |
| Reset All Settings | Restore default appearance/playback settings |
| Recover Quarantined Session | If a previous session failed to load, recover it here |
