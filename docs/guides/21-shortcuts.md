# 21 — Keyboard Shortcuts

Complete list of every keyboard shortcut in Adaptive Session Studio.

---

## Playback

| Key | Action |
|-----|--------|
| `Space` | Play / Pause |
| `Enter` | Stop session |
| `ESC × 2` | Emergency stop (double-press ESC) |
| `F` | Toggle fullscreen |
| `N` | Skip to next scene |
| `Shift` | Toggle FunScript pause (device stops, session continues) |

---

## Navigation

| Key | Action |
|-----|--------|
| `←` | Skip back 10 seconds |
| `→` | Skip forward 10 seconds |
| `↑` | Skip forward 30 seconds |
| `↓` | Skip back 30 seconds |

---

## Live controls

| Key | Action |
|-----|--------|
| `[` | Intensity −10% |
| `]` | Intensity +10% |
| `,` | Speed −0.1× |
| `.` | Speed +0.1× |
| `R` | Reset live controls to defaults |

---

## Macros

| Key | Action |
|-----|--------|
| `1` | Inject macro slot 1 |
| `2` | Inject macro slot 2 |
| `3` | Inject macro slot 3 |
| `4` | Inject macro slot 4 |
| `5` | Inject macro slot 5 |

---

## Editor

| Key | Action |
|-----|--------|
| `Ctrl+Z` | Undo |
| `Ctrl+Y` | Redo |
| `Ctrl+Shift+Z` | Redo (alternative) |
| `Ctrl+S` | Export session as .assp file |
| `Ctrl+,` | Open settings |
| `Ctrl+0` | Reset timeline zoom |
| `Scroll` | Zoom timeline in/out |
| `E` | Toggle FunScript edit mode |
| `A` | Toggle attention tracking on/off |
| `D` | Toggle device connection |
| `Delete` / `Backspace` | Delete selected block (when focused) |

---

## Help

| Key | Action |
|-----|--------|
| `?` | Show shortcut overlay |

---

## Timeline interaction

| Action | How |
|--------|-----|
| Seek during playback | Click anywhere on the progress bar |
| Preview position (paused) | Shift+click on the progress bar |
| Zoom timeline | Scroll over the timeline |
| Reset zoom | Ctrl+0 |
| Select block | Click in sidebar or timeline |
